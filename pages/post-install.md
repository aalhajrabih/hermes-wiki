# 🚀 بعد التثبيت

بعد تثبيت Hermes Agent بنجاح، اتبع هذه الخطوات لتشغيله.

## ⚙ الإعداد الأولي

شغّل معالج الإعداد التفاعلي:

```bash
hermes setup
```

سيطلب منك المعالج:

1. **المزود (Provider)**: OpenAI، Anthropic، OpenRouter، DeepSeek، إلخ
2. **مفتاح API**: المفتاح الخاص بمزودك
3. **النموذج الافتراضي**: النموذج الذي سيُستخدم في المحادثات
4. **تفضيلات الأدوات**: الأدوات التي تريد تفعيلها (terminal، browser، web، إلخ)

## 🤖 اختيار النموذج

استخدم أمر `hermes model` لتغيير النموذج في أي وقت:

```bash
hermes model
```

### المزودات المتاحة

| المزود | المميزات | التكلفة |
|--------|----------|---------|
| **Nous Portal** ★ | نماذج Hermes الأصلية، بدون مفتاح API | مجاني |
| **OpenRouter** | 200+ نموذج من مزودين متعددين | ادفع حسب الاستخدام |
| **OpenAI / Anthropic** | GPT-4o، Claude Sonnet 4 | بمفاتيح API الخاصة بك |
| **DeepSeek** | DeepSeek-V3، R1 | اقتصادي |
| **Ollama محلي** | شغّل النماذج محلياً على هاتفك | مجاني |

!!! توصية
للمبتدئين: ابدأ بـ **Nous Portal** (مجاني، لا يحتاج مفتاح). للمستخدمين المتقدمين: **Anthropic Claude** يعطي أفضل أداء للمهام المعقدة.

## 💬 بدء المحادثة (CLI)

لبدء محادثة مباشرة من الطرفية:

```bash
hermes
```

يفتح واجهة طرفية متكاملة مع:

- تحرير متعدد الأسطر
- إكمال تلقائي
- مخرجات أدوات متدفقة
- تاريخ المحادثات

## 📱 بوابة تيليغرام

لربط هيرميس مع تيليغرام والتحكم فيه عن بعد:

```bash
hermes gateway setup
hermes gateway run
```

### إنشاء بوت تيليغرام

1. تحدث مع [@BotFather](https://t.me/BotFather) على تيليغرام
2. أرسل `/newbot` واتبع التعليمات
3. انسخ **توكن البوت** (مثل `123456789:ABCdef...`)
4. الصقه في `hermes gateway setup`

:::warn تنبيه
إذا كنت تشغّل **OpenClaw أو وكيلاً آخر**، استخدم توكن بوت تيليغرام مختلف لكل وكيل.
:::

## 🔄 التعايش مع OpenClaw

إذا كنت تستخدم أكثر من وكيل على نفس الجهاز:

| الوكيل | البوت | الطريقة |
|--------|-------|---------|
| **Hermes** | بوت #1 | `hermes gateway setup` |
| **OpenClaw** | بوت #2 | اضبط في إعدادات OpenClaw بتوكن مختلف |

:::success بدون تعارض
مع توكنات منفصلة، يعمل كلا الوكيلين معاً دون تعارض.
:::
